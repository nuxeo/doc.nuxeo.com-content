## Bugfixes

###  Resolved Client Connect Document Creation page
Resolved a key production issue related to missing fields during Client Connect document creation. This improves application stability and enhances accuracy for end users.

**What Changed:** 
Added the same configuration options that were available in the JSF view for creating Connect documents.

**Impact:**
Users now have all the necessary options to configure Connect documents, achieving feature parity with the previous JSF interface.
